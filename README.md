# Student Grade Calculator

A beginner-friendly Python program that collects student scores, calculates an average, and assigns a letter grade.

## Features
- Accepts a student name with text cleaning
- Accepts multiple subject scores
- Converts inputs to numeric values for accuracy
- Stores scores in a dictionary (JSON-style structure)
- Calculates the average score
- Automatically assigns a letter grade using conditional logic

## Version Notes
- v1: Manual grade selection
- v2: Automated grading with if / elif / else logic

This version intentionally avoids loops, functions, and error handling to remain aligned with the current learning stage.

## How to Run
```bash
python grade_calculator.py
